# Apicentric (NPM Wrapper)

> A powerful CLI tool and API simulator platform for developers who love the terminal.

This package is a Node.js wrapper for the [Apicentric CLI](https://github.com/pmaojo/apicentric). It automatically downloads the correct binary for your system (Windows, macOS, Linux) and allows you to run it via `npm` or `npx`.

## Installation

### Global Install

```bash
npm install -g apicentric
```

### Run via NPX (No install required)

```bash
npx apicentric simulator start
```

## Usage

Once installed, use the `apicentric` command:

```bash
apicentric --help
apicentric doctor
apicentric simulator start
```

For full documentation, visit the [Main Repository](https://github.com/pmaojo/apicentric).
