# **App Name**: Apicentric REST API

## Core Features:

- Simulator Start/Stop: Start and stop the Apicentric simulator via API calls, including options for service directory, force, and P2P mode.
- Service Definition Management: Manage service definitions including validating, importing, exporting, creating new definitions (YAML or GraphQL), and editing endpoints.
- AI Service Definition Generation: Use AI to generate service definitions based on natural language prompts.
- Log Retrieval: Retrieve simulator logs with filtering options (service, method, route, status) and output formats (JSON).
- Client Code Generation: Generate TypeScript types, React Query hooks, and default React components from service definitions.
- Simulator Status Monitoring: Provide detailed simulator status and individual service status with a detailed view including endpoints and configuration.

## Style Guidelines:

- Primary color: Soft blue (#90AFC5) to evoke a sense of reliability and calm control.
- Background color: Light gray (#F0F4F7), nearly white, for a clean and modern interface.
- Accent color: Muted green (#A9D08E) for highlighting interactive elements and success states.
- Body and headline font: 'Inter', a sans-serif font providing a clean and modern feel, suitable for both headlines and body text.
- Use simple, outline-style icons to represent different simulator functions and operations.
- A clean, well-organized layout with clear visual hierarchy to facilitate easy navigation and control of the Apicentric simulator.
- Subtle animations for loading states and transitions to provide visual feedback without being distracting.
